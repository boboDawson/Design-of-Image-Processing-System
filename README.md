# 图像处理系统 (Image Processing System)

## 项目简介
这是一个基于 Python 和 Tkinter 开发的图像处理桌面应用程序。系统集成了用户登录/注册、图像编辑、滤镜效果、操作历史记录以及日志追踪等功能，旨在提供一个功能丰富且易于扩展的图像处理平台。

## 功能特性
### 👤 用户系统
- **安全验证**：支持用户注册、登录，密码经过 SHA-256 加密存储。
- **状态管理**：通过文件记录登录状态，避免重复登录，支持用户切换与退出。

### 🖼️ 图像编辑功能
- **基础 IO**：支持打开、保存图片（JPEG、PNG、BMP 等主流格式）。
- **图像变换**：裁剪、旋转、翻转图像。
- **创意绘制**：自由绘图（支持颜色/粗细选择），添加文字水印（支持字体大小、颜色、粗细自定义）。
- **特效滤镜**：反相、黑白、风格化、素描、浮雕、复古、模糊、二值化等。
- **色彩调整**：色阶调整，独立调节亮度、对比度与 RGB 通道。
- **形态学操作**：边缘检测、腐蚀、膨胀等高级图像处理。

### 📜 日志与记录
- **操作历史**：完整记录每一步图像处理操作，支持撤销与重置。
- **日志追踪**：记录所有用户操作（登录、打开图片、应用效果等），同步写入文件与数据库。

### 💾 数据存储
- 使用 SQLite 轻量级数据库，稳定存储用户信息、图片上传记录和处理日志。

## 技术栈
| 类别 | 技术选型 |
| :--- | :--- |

## 安装与运行

### 环境要求
- Python 3.6 或更高版本
- 操作系统：Windows / macOS / Linux (兼容所有主流系统)

### 安装依赖
打开命令行终端，运行以下命令安装所需第三方库：
```bash
pip install opencv-python numpy Pillow
```
注意：Tkinter、sqlite3、hashlib、logging 等库为 Python 标准库，无需额外安装。
运行方式
克隆 / 下载：将本项目克隆或下载到本地电脑。
进入目录：打开终端，切换到项目根目录。
启动程序：
bash
运行
# 方式一：启动主程序入口
python main.py

# 方式二：若主程序不可用，直接启动登录系统
python login_system.py
首次运行：程序会自动创建 logs 文件夹用于存放日志，并生成 image_editor.db 数据库文件。
### 模块说明
| 文件名 | 功能说明 |
|:--- |:--- |
| main.py | 程序总入口，负责检查登录状态，并根据状态启动登录界面或主界面 |
| login_system.py | 登录与注册界面模块，处理用户验证、注册逻辑与登录状态保存 |
| image_editor.py | 图像编辑器核心模块，包含所有图像处理逻辑、UI 布局和历史记录管理 |
| db_manager.py | 数据库管理模块，封装 SQLite 的连接、查询及增删改查操作 |
| login_interceptor.py | 登录状态拦截器，通过读取 `login_status.txt` 实现自动登录功能 |
| logger_config.py | 日志配置模块，设置日志格式，将输出同时写入控制台和按日期滚动的日志文件 |
| utils.py | 通用工具模块，包含密码复杂度验证、格式校验等辅助功能函数 |

### 使用指南
#### 首次使用
1. 运行程序后进入登录界面。
2. 点击「注册」按钮，创建一个新账户（需满足密码复杂度要求）。
3. 注册成功后，输入用户名和密码，点击「登录」进入主界面。

#### 图像编辑流程
1. **打开图片**：点击左侧工具栏的「打开图片」按钮，选择本地图片。
2. **基础操作**：使用裁剪、旋转、文字、绘图等工具编辑图片。
3. **应用特效**：点击右侧的滤镜按钮（如黑白、模糊、素描等）预览效果。
4. **历史管理**：通过「应用」确认步骤，或「取消」回退；点击「重置所有」可撤销全部步骤。

#### 用户管理
- **切换用户**：点击顶部用户面板的「切换用户」，系统会清除当前状态并返回登录界面。
- **退出登录**：点击「退出登录」，安全退出当前账户。
- **退出程序**：直接关闭主窗口即可退出程序。

## 数据库结构
### users (用户信息表)
| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| user_id | INTEGER | 用户ID (主键，自增) |
| username | TEXT | 用户名 (唯一) |
| password | TEXT | SHA-256 加密后的密码 |
| email | TEXT | 用户邮箱 |
| create_time | TEXT | 账户创建时间 |
| last_login | TEXT | 最后一次登录时间 |

### images (图片记录表)
| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| image_id | INTEGER | 图片ID (主键，自增) |
| user_id | INTEGER | 关联的用户ID |
| filename | TEXT | 图片原始文件名 |
| filepath | TEXT | 图片本地存储路径 |
| format | TEXT | 图片格式 (JPG/PNG等) |
| size | INTEGER | 图片大小 (字节) |
| upload_time | TEXT | 上传时间 |

### process_logs (处理日志表)
| 字段名 | 类型 | 说明 |
| :--- | :--- | :--- |
| log_id | INTEGER | 日志ID (主键，自增) |
| image_id | INTEGER | 关联的图片ID |
| user_id | INTEGER | 操作用户ID |
| process_type | TEXT | 操作类型 (如"滤镜-黑白") |
| parameters | TEXT | 操作参数 (JSON格式) |
| process_time | TEXT | 处理时间 |

## 注意事项
1. **密码安全**：密码以 SHA-256 哈希值存储，不可逆，请务必牢记密码。
2. **权限问题**：请确保项目目录有读写权限，以便程序正常创建 `logs` 文件夹和数据库文件。
3. **数据备份**：图像处理操作会实时记录到数据库，请勿随意删除 `image_editor.db` 文件。
4. **日志文件**：日志文件存储在 `logs` 目录下，按日期分割，可用于排查运行问题。
